# The National Aeronautics and Space Administration (national-aeronautics-and-space-administration)

NASA explores the unknown in air and space, innovates for the benefit of humanity, and inspires the world through discovery. The api.nasa.gov portal hosts a federated set of APIs that make NASA imagery, science, and mission data accessible to application developers.

## APIs

### APOD - Astronomy Picture of the Day
The Astronomy Picture of the Day, exposed as JSON metadata and image links.
- **Base URL:** `https://api.nasa.gov/planetary/apod`
- [Documentation](https://api.nasa.gov/) | [SourceCode](https://github.com/nasa/apod-api)

### NeoWs - Near Earth Object Web Service
RESTful service for near earth Asteroid information including approach data and orbital data.
- **Base URL:** `https://api.nasa.gov/neo/rest/v1`
- [Documentation](https://api.nasa.gov/) | [SourceCode](https://github.com/SpaceRocks/NeoWs)

### DONKI - Space Weather Database Of Notifications, Knowledge, Information
Comprehensive online tool for space weather forecasters, scientists, and researchers.
- **Base URL:** `https://api.nasa.gov/DONKI`
- [Documentation](https://api.nasa.gov/)

### Earth Imagery and Assets
Earth imagery API providing Landsat 8 imagery and asset metadata.
- **Base URL:** `https://api.nasa.gov/planetary/earth`
- [Documentation](https://api.nasa.gov/)

### EONET - Earth Observatory Natural Event Tracker
Curated source of continuously updated natural event metadata.
- **Base URL:** `https://eonet.gsfc.nasa.gov/api/v3`
- [Documentation](https://eonet.gsfc.nasa.gov/docs/v3)

### EPIC - Earth Polychromatic Imaging Camera
Full disc imagery of the Earth captured by the DSCOVR spacecraft.
- **Base URL:** `https://api.nasa.gov/EPIC/api`
- [Documentation](https://epic.gsfc.nasa.gov/about/api)

### Mars Rover Photos
Image data gathered by NASA's Mars rovers.
- **Base URL:** `https://api.nasa.gov/mars-photos/api/v1`
- [Documentation](https://api.nasa.gov/) | [SourceCode](https://github.com/chrisccerami/mars-photo-api)

### NASA Image and Video Library
The public NASA media library API.
- **Base URL:** `https://images-api.nasa.gov`
- [Documentation](https://images.nasa.gov/docs/images.nasa.gov_api_docs.pdf)

### TLE - Two Line Element Set
Up to date two line element set records for earth-orbiting objects.
- **Base URL:** `https://tle.ivanstanojevic.me/api`
- [Documentation](https://tle.ivanstanojevic.me/)

### Exoplanet Archive API
Programmatic access to NASA's Exoplanet Archive.
- **Base URL:** `https://exoplanetarchive.ipac.caltech.edu/TAP`
- [Documentation](https://exoplanetarchive.ipac.caltech.edu/docs/program_interfaces.html)

### InSight - Mars Weather Service
Per-Sol summary data from the InSight lander on Mars.
- **Base URL:** `https://api.nasa.gov/insight_weather`
- [Documentation](https://api.nasa.gov/)

### TechPort
NASA technology project data in a machine readable format.
- **Base URL:** `https://api.nasa.gov/techport/api`
- [Documentation](https://techport.nasa.gov/api)

### SSD/CNEOS
Solar System Dynamics and Center for Near-Earth Object Studies resources.
- **Base URL:** `https://ssd-api.jpl.nasa.gov`
- [Documentation](https://ssd-api.jpl.nasa.gov/doc/)
